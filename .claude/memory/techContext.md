**Стэк технологий:**
- React 18.3.1
- Vite 6.0.5 + TypeScript 5.9.3
- Tailwind CSS 3.4.17
- PostCSS 8.4.49 + Autoprefixer 10.4.20

**Ключевые библиотеки:**
- react-router-dom 7.13.1 — роутинг
- framer-motion 11.15.0 — анимации
- swiper 12.1.2 — слайдеры/карусели
- flubber 0.4.2 — работа с состоянием

**Команды:**
- `npm run dev` — запуск dev-сервера
- `npm run build` — сборка для продакшена
- `npm run preview` — предпросмотр сборки

**Конфигурация:** vite.config.ts, tsconfig.json, tailwind.config.js, postcss.config.js
