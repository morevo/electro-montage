**Архитектура проекта:**

Стандартная Vite + React структура:
- `src/` — исходный код компонентов, страниц, стили
- `public/` — статические ассеты (изображения, favicon)
- `dist/` — скомпилированная сборка

**Паттерны:**
- Функциональные компоненты React (вероятно, используется hooks API)
- Router для навигации между страницами
- Tailwind для стилизации без отдельного CSS файла
